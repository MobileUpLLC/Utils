## Installation

Utils doesn't contain any external dependencies.

### CocoaPods

1. Make ```pod init``` 
2. Add the following to Podfile 

```
pod 'Utils', :git => 'https://github.com/MobileUpLLC/Utils', :tag => '0.0.1'
```

3. Make ```pod install```

### Manual

Download and drag files from Source folder into your Xcode project.

### Swift Package Manager Install

Swift Package Manager 

```
dependencies: [
    .package(url: "https://github.com/MobileUpLLC/Utils", .upToNextMajor(from: "0.0.1"))
]
```
